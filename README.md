# Animal Guardians 動物守護者

兒童合作闘關 RPG 桌遊 - Print & Play 版本

## 遊戲資訊
- **玩家人數**：2-3 人（1 DM + 2 小玩家）
- **適合年齡**：6-12 歲
- **遊戲時間**：15-20 分鐘
- **類型**：合作、擲骰、角色扮演

## 快速開始

### 需要準備
- 2 顆六面骰（2d6）
- 5 個愛心標記（硬幣、糖果皆可）
- 列印好的動物卡（見下方）

### 列印說明
1. 開啟 `animal-cla-2.md`（最終版本）
2. 參考 Part 4 版面指引
3. A4 紙列印，建議 160gsm 以上
4. 裁切成 63×88mm 卡片
