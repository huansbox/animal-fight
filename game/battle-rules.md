# 動物大對決 - 淘汰賽規則

> **遊戲人數**：2 人
> **遊戲時間**：5-15 分鐘（視參賽動物數量）
> **配件需求**：動物卡 + 2 顆六面骰

---

## 賽制

採用**淘汰賽**形式（類似世界盃足球賽），每輪 **3 戰 2 勝制**。

```
範例：8 強淘汰賽

    八強賽        四強賽        決賽

    非洲獅 ─┐
            ├─ 勝者 ─┐
    穿山甲 ─┘        │
                     ├─ 勝者 ─┐
    獵豹 ───┐        │        │
            ├─ 勝者 ─┘        │
    貓頭鷹 ─┘                  │
                               ├─ 冠軍！
    科摩多龍─┐                 │
            ├─ 勝者 ─┐        │
    海豚 ───┘        │        │
                     ├─ 勝者 ─┘
    大猩猩 ─┐        │
            ├─ 勝者 ─┘
    蜜獾 ───┘
```

---

## 對戰流程

### 每一戰的流程

1. **雙方各擲 2 顆骰子**
2. **計算各自分數**（見下方計分規則）
3. **比較分數**，高者獲勝
4. **平手則加賽**（再骰一輪）

### 每一輪的流程

- 先取得 **2 勝** 者晉級
- 最多進行 3 戰

---

## 計分規則

### 基本計分

擲出的 2 顆骰子，各自對應 1 個屬性，將兩個屬性數值相加。

| 骰子點數 | 對應屬性 |
|:--------:|----------|
| 1 | 力量 |
| 2 | 速度 |
| 3 | 攻擊 |
| 4 | 防禦 |
| 5 | 智慧 |
| 6 | 天賦（特殊能力觸發）|

**範例**：蜜獾骰出 2 + 4
- 2 = 速度 (6)
- 4 = 防禦 (7)
- 總分 = 6 + 7 = **13 分**

---

### 特殊情況

#### 情況 1：骰出相同點數（非雙 6）

該屬性計算 **2 次**。

**範例**：蜜獾骰出 3 + 3
- 3 = 攻擊 (7)
- 總分 = 7 + 7 = **14 分**

---

#### 情況 2：骰出 1 個 6

**特殊能力觸發！** 另一顆骰子對應的屬性獲得加成。

**範例**：蜜獾骰出 4 + 6
- 4 = 防禦 (7)
- 6 = 特殊能力「百毒不侵」觸發
- 蜜獾的特殊加成：防+3, 智+1
- 防禦獲得 +3 加成
- 總分 = 7 + 3 = **10 分**

**範例**：蜜獾骰出 5 + 6
- 5 = 智慧 (4)
- 6 = 特殊能力觸發
- 智慧獲得 +1 加成
- 總分 = 4 + 1 = **5 分**

> **注意**：特殊能力只加成特定屬性，骰到其他屬性可能沒有加成！

---

#### 情況 3：骰出雙 6

**重骰 1 次**，重骰後無論結果如何，特殊能力都會觸發。

**範例**：蜜獾骰出 6 + 6
1. 重骰 → 得到 2 + 4
2. 2 = 速度 (6)，4 = 防禦 (7)
3. 特殊能力觸發：速度無加成，防禦 +3
4. 總分 = 6 + (7 + 3) = **16 分**

**範例**：蜜獾骰出 6 + 6 → 重骰 → 3 + 5
1. 3 = 攻擊 (7)，5 = 智慧 (4)
2. 特殊能力觸發：攻擊無加成，智慧 +1
3. 總分 = 7 + (4 + 1) = **12 分**

> **重骰規則**：
> - 重骰最多 1 次
> - 重骰後若又骰到 6，視為普通結果（特殊能力已觸發）
> - 重骰後若又骰到雙 6，不再重骰，兩顆都套用特殊加成

## 完整對戰範例

### 蜜獾 vs 獵豹

**【第一戰】**
```
蜜獾擲骰：4 + 6
  → 防禦 (7) + 特殊加成 (+3) = 10 分

獵豹擲骰：2 + 3
  → 速度 (9) + 攻擊 (6) = 15 分

結果：獵豹勝
```

**【第二戰】**
```
蜜獾擲骰：3 + 3
  → 攻擊 (7) × 2 = 14 分

獵豹擲骰：1 + 5
  → 力量 (4) + 智慧 (4) = 8 分

結果：蜜獾勝
```

**【第三戰】**
```
蜜獾擲骰：6 + 6 → 重骰 → 2 + 4
  → 速度 (6) + 防禦 (7+3) = 16 分

獵豹擲骰：2 + 6
  → 速度 (9) + 特殊加成 (+2) = 11 分

結果：蜜獾勝
```

**最終結果：蜜獾 2:1 獲勝晉級！**

## 進階玩法（選用）

### 選角階段

比賽開始前，雙方輪流從動物池中選擇參賽動物：
1. 決定參賽動物數量（4、8、16 隻）
2. 擲骰決定先選權
3. 輪流選擇，直到雙方動物數量相等
4. 抽籤決定對戰組合

### 禁用規則

為增加變化性，可設定「禁用」：
- 每人可禁用對方 1 隻動物
- 被禁用的動物本場比賽不能參加

---

## 快速開始

1. 各選 4 隻動物
2. 抽籤配對（或自行決定）
3. 開打！3 戰 2 勝晉級
4. 最後 1 隻存活者為冠軍
