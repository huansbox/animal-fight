# 注音字體 prototype 子集

本目錄只供 `card/bopomofo-card-prototype.html` 比較字體使用。四個 WOFF2 檔案皆由官方 v1.500 TTF 依 prototype 用字縮減，不是正式卡片完整字型；正式版需改用完整字型或依全部動物名稱重新建立子集。

| Prototype 檔案 | 原始字體 | 官方來源 |
|---|---|---|
| `rounded-bold-subset.woff2` | 源泉注音圓體 Bold | <https://github.com/ButTaiwan/bpmfvs> |
| `rounded-medium-subset.woff2` | 源泉注音圓體 Medium | <https://github.com/ButTaiwan/bpmfvs> |
| `gothic-bold-subset.woff2` | 源石注音黑體 Bold | <https://github.com/ButTaiwan/bpmfvs> |
| `gothic-medium-subset.woff2` | 源石注音黑體 Medium | <https://github.com/ButTaiwan/bpmfvs> |

字體依 SIL Open Font License 1.1 使用，完整授權與著作權聲明見同目錄的 `LICENSE-Gen.txt`。
