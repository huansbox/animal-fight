# 動物守護者黑白圖示系統

## 核心規格

- SVG `viewBox="0 0 64 64"`
- 主線寬 `4.5`，圓角端點與圓角轉折
- 純黑線、白底，不使用灰階與陰影
- 屬性圖示以輪廓為主；骰點使用純黑填色
- 實際印刷建議：屬性圖示 8–10mm、骰面 4.5–6mm
- 所有圖示搭配文字或明確版面位置，不作為唯一資訊來源

## 已完成

- `attribute-strength.svg`：力量／槓鈴
- `attribute-speed.svg`：速度／跑鞋與速度線
- `attribute-attack.svg`：攻擊／交叉雙劍
- `attribute-defense.svg`：防禦／盾牌
- `attribute-smart.svg`：聰明／大腦
- `die-1.svg` ～ `die-6.svg`：骰面 1–6
