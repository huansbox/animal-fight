# 動物圖片黑白印刷風格測試

本目錄用同一組四隻動物、相同動作與構圖，比較三種黑白插畫細節量。目標輸出是 114×70mm 動物卡圖片框，使用黑白雷射印表機列印。

## 定案

2026-07-19 試印後，非洲象、黑豹、河豚、行軍蟻全部選擇 `v2`。後續 132 張正式動物圖片統一使用 V2「圖鑑平衡版」；`v1`、`v3` 保留為比較紀錄。

## 測試動物

- 非洲象：鼻子噴出清楚的弧形水流
- 黑豹：從低處向前撲躍
- 河豚：鼓起身體、短刺展開
- 行軍蟻：沿落葉旁形成前進隊伍

## 三種版本

- `v1`：五階灰階、較細外輪廓、保留少量自然細節
- `v2`：五階灰階、中粗外輪廓、只保留辨識物種必要的紋理
- `v3`：四階灰階、粗外輪廓、幾乎不使用表面紋理

三個版本只應改變灰階層次、外輪廓粗細與紋理密度，不應改變動物姿勢、特殊能力動作或背景配置。

## 比較方式

1. 開啟 `../animal-image-style-3x4-comparison.html`，或列印對應 PDF。
2. 列印設定使用 A4 橫向、100%／實際大小、關閉頁首頁尾。
3. 先看 5 歲孩子能否說出動物與正在做的事，再比較輪廓、灰階是否糊成一片。
4. 正式圖片使用已選定的 V2，以同一套 prompt 規格批次產生 132 張。

完整 prompt 與 QA 條件見 [`../img-prompt/animal-print-bplus-3x4.md`](../img-prompt/animal-print-bplus-3x4.md)。
