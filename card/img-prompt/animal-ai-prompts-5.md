### 修正項目

| 動物 | 原問題 | 修正內容 |
|------|--------|----------|
| 虎鯨 | 場景含冰上獵物過於複雜；顏色描述無效 | 移除場景，改用「eye patch marking」結構特徵 |
| 雪豹 | 跳躍姿態與花豹撲殺難以區分 | 改為伏低蹲姿（crouching），強調超長粗尾巴 + 寬掌 |
| 白頭海鵰 | 俯衝姿態與遊隼重疊 | 改為水面低掠抓魚（horizontal），與遊隼垂直俯衝區隔 |
| 雪鴞 | 與貓頭鷹輪廓相似；顏色描述無效 | 改水平滑翔姿態，明確標注「without ear tufts」+ 厚羽覆腿 |
| 乳牛 | 缺乏辨識特徵，可能誤認為牛科其他動物 | 新增「patchy markings」+「prominent udder」 |
| 擬態章魚 | 指定獅子魚/比目魚等擬態目標過於複雜 | 簡化為觸手各展不同形狀，呈現變形概念 |
| 皇帝企鵝 | 顏色描述無效（黑白色塊在線稿中無意義） | 改為冰緣跳水姿態 + 氣泡尾跡 |
| 犰狳 | 捲球姿態與穿山甲相似 | 強化「rectangular bands」矩形甲片質感，與穿山甲鱗片區隔 |
| 孔雀 | 開屏描述不夠具體 | 新增「semicircular」+「large bold eye-spot」+ 纖細體型 |
| 箭毒蛙 | 「vivid」為顏色描述，線稿無效 | 移除顏色詞，改用幾何警戒花紋 + 抬腿蓄力姿態 |
| 蘭花螳螂 | 花瓣背景增加場景複雜度 | 移除背景花朵，單獨呈現螳螂花形輪廓 |
