動物卡視覺方向比較

所有圖片框與屬性列皆使用實際卡面尺寸：圖片 114×70mm、屬性列 114×40mm。

一、動物圖片細節量

同一隻非洲象、同一個「鼻子噴水」動作，只改繪製方式。C 的水流方向差異來自生成隨機性，不是刻意設計；正式 prompt 會固定動作朝向。

A｜現有高寫實 皮膚與地面細節最多；螢幕效果好，黑白雷射最可能糊成灰。
B｜簡化寫實插畫 灰階區塊清楚、耗碳粉少；可能稍微接近插畫感，作為簡化邊界。
我較推薦
C｜高對比自然史版畫 保留寫實輪廓，以黑、灰、白和較粗線條呈現；預期雷射列印最穩。

二、屬性 icon 與骰子位置

骰面 1–5 已統一置中放在 icon 正上方；三個版本只比較 icon 風格。

目前建議：骰子固定放 icon 正上方；icon 優先測 H「具象線面混合」，兼顧孩子辨識與較成熟的黑白造型。圖片先保留 A、B、C，等 icon 定案後一起做實際雷射列印。
