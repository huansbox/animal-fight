> 日期：2026-04-08
> 狀態：草案，待 review
> 目標受眾：5 歲 + 9 歲男孩
> 載體：雷射黑白列印，A5 大小卡牌

---

## 共用設定

### 動物卡基本資料（沿用現有 132 隻）

每張動物卡包含：
- **五維屬性**：力量(💪)、速度(⚡)、攻擊(⚔)、防禦(🛡)、智慧(🧠)，數值 1-9
- **HP = 五維總和 × 2**（讓數字更大、戰鬥回合更多、更有「打架」的感覺）
- **特殊能力**：技能名稱 + 加成效果（固定 +4 分配到 2-3 屬性）
- **棲地類型**：陸 🏔 / 空 🦅 / 海 🌊（新增，用於相剋）

### 棲地分類原則（132 隻）

| 棲地 | 定義 | 代表動物 |
|------|------|---------|
| 陸 🏔 | 主要在地面活動 | 獅子、象、獵豹、穿山甲、灰狼 |
| 空 🦅 | 會飛或主要在空中 | 貓頭鷹、白頭海鵰、角鵰、螢火蟲 |
| 海 🌊 | 水棲或半水棲 | 虎鯨、大白鯊、海豚、灣鱷、皇帝企鵝 |

### 棲地相剋

```
陸 🏔 → 剋 空 🦅（弓箭手打鳥：地面伏擊降落的飛行動物）
空 🦅 → 剋 海 🌊（魚鷹抓魚：俯衝捕獵水面獵物）
海 🌊 → 剋 陸 🏔（鱷魚拖岸邊：水中拖入岸邊動物）
```

- 剋制方：攻擊傷害 +3
- 被剋方：無額外減免（保持簡單）
- 同棲地：無加成

## Layer 1：動物對戰（5 歲能玩版）

### 設計目標
- 5 分鐘學會，8-10 分鐘一局
- 每回合只需做 1 個決策
- 保留擲骰的刺激感
- 無需閱讀大量文字

### 準備

| 項目 | 規格 |
|------|------|
| 牌組 | 每人 8 張動物卡 |
| 手牌 | 開局抽 5 張 |
| 場地 | 戰鬥區 1 隻 + 備戰區最多 2 隻 |
| 勝利 | 先擊倒對手 3 隻動物獲勝 |
| 道具 | 2 顆六面骰、HP 計數器（骰子/紙筆/代幣皆可） |

### 開局
1. 各自洗牌，抽 5 張手牌
2. 各選 1 隻放戰鬥區（面朝下），同時翻開
3. 可選擇放 0-2 隻到備戰區（面朝上）
4. 剩餘手牌為牌組（不再抽牌，簡化）

### 回合流程（先攻丟銅板決定）

```
我的回合：
  ① 可選：把戰鬥區動物換成備戰區的（免費換，每回合限 1 次）
  ② 必做：擲 2 顆骰子 → 計算傷害 → 扣對手戰鬥區動物的 HP
  → 換對手回合
```

### 傷害計算（沿用現有骰子系統，簡化版）

骰子點數對應屬性：
- 1 = 力量、2 = 速度、3 = 攻擊、4 = 防禦、5 = 智慧
- **6 = 特殊能力！**（重骰該顆 + 觸發技能加成）

**基本傷害** = 兩顆骰子對應的屬性值相加

**骰到 6 時**：
1. 該顆重骰（得到新點數 1-5 的屬性值）
2. 如果新點數對應的屬性有技能加成 → 額外加上加成值
3. 再骰到 6？再重骰一次！（可累加，小孩超愛）

**棲地加成**：若我方剋制對手 → 總傷害 +3

**骰出相同點數**：該屬性 ×2（雙倍！）

### HP 系統

HP = 五維總和 × 2

| 動物 | 五維總和 | HP |
|------|:-------:|:--:|
| 虎鯨 | 33 | 66 |
| 非洲獅 | 30 | 60 |
| 獵豹 | 25 | 50 |
| 貓頭鷹 | 19 | 38 |
| 穿山甲 | 17 | 34 |
| 螢火蟲 | 10 | 20 |

HP 歸零 → 該動物被擊倒 → 移到「墓地」
對手從備戰區或手牌派新動物上場

**第 1 回合 — A 的回合**

場面：非洲獅(陸) vs 虎鯨(海)
棲地：陸 vs 海 → **虎鯨剋獅子？不對，海剋陸，但獅子在攻擊方**
修正：A 攻擊 B，A 是陸、B 是海 → 海剋陸，但那是 B 剋 A，A 攻擊時 A 不佔優勢。
→ **棲地加成只看攻擊方是否剋防守方**：陸剋空、空剋海、海剋陸
→ A(陸) 攻擊 B(海)：陸不剋海，無加成

A 擲骰：**3, 5**
- 骰 3 = 攻擊 → 獅子攻擊值 = 8
- 骰 5 = 智慧 → 獅子智慧值 = 5

### Layer 1 模擬觀察

| 觀察 | 評價 |
|------|------|
| 一局約 16 回合（8 個完整回合 × 2 人） | 約 10-12 分鐘，長度適中 |
| 每回合決策點只有 1 個（換不換動物） | 5 歲可以理解 |
| 骰 6 觸發特殊能力時很興奮 | 保留了爆發爽感 |
| 棲地相剋 +3 讓換動物有意義 | 簡單策略層 |
| HP 數字偏大（60-66），計算較繁瑣 | ⚠️ 5 歲可能需要幫忙算 |
| 沒有「抽牌」環節 | 簡化成功，但少了期待感 |
| 弱動物（螢火蟲 HP 20）幾乎不會被選 | ⚠️ 低數值動物缺少存在感 |

## Layer 2：動物爭霸（9 歲有深度版）

### 與 Layer 1 的差異

| 項目 | Layer 1 | Layer 2 |
|------|---------|---------|
| 牌組大小 | 8 張動物卡 | 12 張動物卡 + 8 張道具卡 = **20 張** |
| 手牌 | 5 張（不再抽牌） | 5 張（每回合抽 1 張） |
| 備戰區 | 最多 2 隻 | 最多 3 隻 |
| 勝利條件 | 擊倒 3 隻 | 擊倒 3 隻 或 對手場上無動物 |
| 能量系統 | 無 | 每回合自動 +1 能量 |
| 特殊能力 | 骰 6 自動觸發 | 需花 2 能量「充能」才能啟用（啟用後骰 6 才觸發） |
| 道具卡 | 無 | 每回合可用 1 張 |
| 換動物 | 免費 | 需花 1 能量（逃跑費用） |
| 棲地加成 | +3 | +3（不變） |

### 新增：道具卡（8 張，混入牌組）

| 卡名 | 效果 | 設計意圖 |
|------|------|---------|
| 🩹 急救包 | 回復 15 HP | 延長動物壽命 |
| 🪤 陷阱 | 對手下回合不能換動物 | 鎖定弱勢對手 |
| 🌪 暴風雨 | 本回合空棲地動物傷害 -5 | 環境干擾 |
| 🔥 森林大火 | 本回合陸棲地動物傷害 -5 | 環境干擾 |
| 🧊 結冰 | 本回合海棲地動物傷害 -5 | 環境干擾 |
| 💪 戰鬥糧食 | 本回合我方傷害 +5 | 爆發道具 |
| 🔄 強制交換 | 指定對手換上備戰區的 1 隻 | 拉弱的上來打 |
| 👁 偵查 | 看對手手牌，抽 1 張自己的牌 | 情報 + 手牌補充 |

### 新增：能量系統（TCG Pocket 式自動產能）

- 每回合開始自動獲得 1 能量（用代幣/骰子計數，最多累積 5）
- 用途：
  - **充能（2 能量）**：啟用戰鬥區動物的特殊能力（啟用後，該動物骰 6 才會觸發加成；未充能時骰 6 只重骰、不加成）
  - **撤退（1 能量）**：換動物
  - **超級攻擊（3 能量）**：本回合傷害 ×1.5（取整）
- 不花能量也能正常攻擊（保底行動）

### 回合流程

```
我的回合：
  ⓪ 自動：+1 能量
  ① 抽 1 張牌
  ② 自由階段（任意順序，各限 1 次）：
     - 從手牌放動物到備戰區
     - 花 1 能量換動物
     - 花 2 能量充能（啟用特殊能力）
     - 花 3 能量宣告超級攻擊
     - 使用 1 張道具卡
  ③ 擲 2 顆骰子 → 計算傷害
  → 換對手回合
```

## 設計迭代筆記（模擬過程中的靈感）

### 問題 1：HP 計算太繁瑣

**觀察**：每回合要做兩位數減法（66-19=47），5 歲很困難，9 歲也嫌煩。

**方案 A — HP 刻度條**：卡片上印 HP 刻度條（每格 5 HP），用迴紋針/夾子標記。
→ 優點：視覺化、不用算數
→ 缺點：A5 卡片空間足夠，但 B&W 列印需設計好

**方案 B — 簡化 HP = 五維總和 ÷ 3 取整（約 3-11）**：用小骰子當 HP 計數器
→ 虎鯨 11、獅子 10、螢火蟲 3
→ 傷害也要同步縮小（÷6 取整？），但會讓骰子技能差異變小

**方案 C — 固定 HP 分級**：依總和分 S/A/B/C 四級，各級固定 HP
→ S(28+)=5 HP、A(22-27)=4 HP、B(16-21)=3 HP、C(≤15)=2 HP
→ 傷害固定為 1（普通攻擊）或 2（觸發技能/棲地加成）
→ **最簡化**，5 歲完全能玩，但失去骰子計算的樂趣

**推薦**：Layer 1 用方案 C（最低門檻），Layer 2 用方案 A（HP 刻度條）

### 問題 2：低數值動物沒有存在感

**觀察**：螢火蟲(10)、箭毒蛙(14)、穿山甲(17) 在純數值對戰中毫無競爭力。

**方案 A — 弱者加成**：HP ≤ 30 的動物，骰 6 時技能加成 ×2
→ 螢火蟲「閃光誘惑」原本智+3、防+1，變成智+6、防+2
→ 小動物變成高風險高回報角色

**方案 B — 偵察兵角色**：弱動物當「前鋒」被擊倒時，觸發一次性效果
→ 螢火蟲被擊倒 →「閃瞎」→ 對手下回合傷害 -5
→ 箭毒蛙被擊倒 →「劇毒」→ 對手戰鬥區動物每回合扣 3 HP，持續 2 回合

**方案 C — 進化/合體**：2 隻弱動物可以「合體」成 1 隻強化動物
→ 例：穿山甲 + 蠍子 = 新角色「甲蠍獸」
→ 問題：需要額外設計合體組合，複雜度太高

**推薦**：方案 B（被擊倒觸發效果），與寶可夢「特性」類似，增加策略深度，且弱動物有被選的理由

### 問題 3：Layer 1 需不需要「抽牌」？

**觀察**：Layer 1 開局發 5 張就不再抽牌，少了「下一張是什麼」的期待感。

**方案**：Layer 1 也加「每回合抽 1 張」，但牌組只有 3 張（開局 5 + 牌組 3 = 8 張）
→ 只抽 3 回合就結束，但至少有點驚喜
→ 或者改成：開局 4 張手牌，每回合抽 1 張直到抽完

**推薦**：開局抽 4 張，每回合抽 1 張。簡單增加期待感。

### 問題 4：超級攻擊是否太強？

**觀察**：Layer 2 模擬中，戰鬥糧食(+5) + 超級攻擊(×1.5) 一回合打 24 點，幾乎半血。

**方案**：
- 超級攻擊改為 +50%（無條件）→ 改為「+5 固定加成」（與戰鬥糧食同級，不可疊加）
- 或：超級攻擊和戰鬥糧食不能同回合使用

**推薦**：超級攻擊改為固定 +5，且與戰鬥糧食同回合不可疊加。簡單又好平衡。
