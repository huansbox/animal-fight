## 9. 後援來源、資格、骰池與標記消耗

### 第一次後援：第一關

- 主要白頭海鵰 0；單動物被補位置需要 1。
- 北極狐從待命區第一次出場，智慧骰池 2，A／B 都有資格。

- 北極狐進再挑戰區；本關不能立即再次叫其他後援。

### 第二次後援：第二關

- 支援後仍為象海豹 1、袋熊 0；一般雙動物固定象海豹的 1，被補位置需要 1。
- 北極狐從再挑戰區第二次出場，選速度，骰池 2；A／B 都有資格。

- A：以 1 取代袋熊的 0，成為 `1＋1`；B：把缺少的 1 補上，同樣成為 `1＋1`。
- 第二關成功，消耗第 2 枚標記。
- 北極狐第二次出場後直接進休息區；袋熊原本 0 成功，仍進再挑戰區。

### 標記用完後

- 第三關海象 0 成功時，兩枚後援標記都已消耗。
- 即使待命區仍有動物、再挑戰區也有白頭海鵰與袋熊，規則仍不允許第三次後援。
- DM 不詢問要派誰，只說：「兩枚救援標記都用完了，這關要保留目前的結果。」

### 一般雙動物 `0＋0` 固定查表

- 固定一邊的 0，另一邊即使由一隻後援補到 1，仍只有 `0＋1`。
- 兩個位置都必須改善，A／B 都不能呼叫一隻後援通關。
- 不消耗標記，也不移動任何卡片；此案例只驗證資格表。

## 13. 阻擋問題、非阻擋問題與純觀察

### 阻擋問題

- 無。指定流程不需補規則即可完成。

### 非阻擋問題

1. 兩枚標記用完時，再挑戰區仍可能有多隻動物。5 歲孩子可能問「為什麼放在再挑戰區卻不能再出場」；DM 需說明再挑戰資格仍在，但全隊本局已沒有呼叫次數。

### 純觀察

- 北極狐的實體移動路徑「待命 → 再挑戰 → 完成」能清楚呈現第一次失敗與最後一次出場。

- 兩枚標記讓第三關不能後援的判斷不需查成功表。
- Agent B 的 `0 → 1` 補救無法區分 A／B 難度；需要 Agent A 的高門檻案例才能觀察實際勝負差異。
