## 三套固定劇情

| 順序 | 原型 | 劇情 | 六位置區間 | 建議用途 |
|---:|---|---|---|---|
| 1 | 災難救援 | [`暴雨森林救援隊`](2026-07-17-01-storm-forest-rescue.md) | 剛好／大／小／小／大／剛好 | 第一次真人實玩；危機與工作最直觀 |
| 2 | 調查解謎 | [`月光博物館的失蹤鈴鐺`](2026-07-17-02-mystery-investigation.md) | 小／大／小／剛好／大／剛好 | 測試不靠戰鬥也能使用攻擊與防禦 |
| 3 | 遠征運送 | [`雲頂村的暖光石`](2026-07-17-03-expedition-delivery.md) | 大／剛好／小／小／大／剛好 | 測試同一目標跨四關持續推進的故事感 |

每套都固定使用單動物、雙動物、單動物、雙動物 Boss，共六個主要位置；小格、剛好格、大格各兩個。

## 第一次實玩順序

1. 先玩「暴雨森林救援隊」，只使用後援 A「取較高、不相加」。
2. 記錄逐關選角時間、三區理解、兄弟協調、後援消耗與 Boss 結果。
3. 同步在紙上記下若採後援 B「補足缺口」是否會改變資格或勝負，但不要在同一局教第二套算法。
4. 若第一局流程可理解，再換「月光博物館」確認玩法不是只適合救災題材。
5. 三套固定劇情都能順暢遊玩後，才評估三類隨機任務卡。

## 批次新增流程

1. 複製共用模板，不先從自由故事開始寫。
2. 先填六位置工作、區間與兩個屬性，確認三種區間各兩格。
3. 再填開場 Boss 提示、逐關揭露、每個屬性的兩個卡住提示。
4. 補齊單邊成功、`0＋0`、後援 A、再挑戰、不回溯與兩種 Boss 結局。
5. 由規則、5 歲理解、兄弟協調三個視角交叉審查。
6. 真人實玩後只依具體觀察微調，不先增加新機制。
