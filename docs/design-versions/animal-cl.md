# 動物守護者（Animal Guardians）— MVP 遊戲設計文件

> 合作闖關 RPG，適合 6-12 歲兒童與家長共玩

---

## Part 1: 給 DM（家長）的規則摘要卡

### 遊戲準備
- 玩家各選 1 張動物卡
- 放置 5 顆愛心標記（共享生命值）
- 準備 2 顆六面骰（2d6）

### 遊戲流程
1. **DM 描述情境** → 擲 d6 查開局情境表
2. **玩家宣告行動** → 選擇用哪隻動物、哪個屬性應對
3. **擲骰判定** → 2d6 + 對應屬性數值
4. **結算結果** → 成功/失敗，調整生命值或獲得獎勵
5. **重複 2-4** → 直到通關或生命歸零

### 判定規則

| 結果 | 條件 | 效果 |
|------|------|------|
| 大成功 | 骰出雙 6 | 直接勝利，獲得 2 前進標記 |
| 成功 | 總和 ≥ 12 | 通過挑戰，獲得 1 前進標記 |
| 失敗 | 總和 < 12 | 扣 1 愛心 |
| 大失敗 | 骰出雙 1 | 直接失敗，扣 2 愛心 |

### 優勢/劣勢判定
- **優勢**（動物天性契合情境）：擲 3d6 取高 2 顆
- **劣勢**（明顯不適合）：擲 3d6 取低 2 顆
- DM 可依情境自由裁決，鼓勵孩子說明選擇理由

### 屬性對照表（骰子點數 = 屬性槽位）

| 骰面 | 圖示 | 屬性 | 適用情境 |
|------|------|------|----------|
| 1 | 拳頭 | 力量 | 推開障礙、搬運重物 |
| 2 | 鞋子 | 速度 | 追逐、閃躲、搶先行動 |
| 3 | 劍 | 攻擊 | 主動進攻、破壞物品 |
| 4 | 盾 | 防禦 | 承受傷害、保護同伴 |
| 5 | 大腦 | 智慧 | 解謎、溝通、判斷陷阱 |
| 6 | 星星 | 天賦 | 發動專屬技能 |

### 終局模式
累積 **5 個前進標記** 後進入終局，二選一：
- **Boss 戰**：連續 3 輪高難度檢定（難度 14）
- **逃脫模式**：擲 d6 決定障礙 → 選隊員檢定 → 連續 3 關

```

├────────────────────────┤

│                        │

├────────────────────────┤

│   大地，誰都推不倒我」 │ ← 角色台詞（增加代入感）
└────────────────────────┘
```

### 屬性槽位設計要點

1. **骰子圖示優先**：使用實際骰子點數圖（⚀⚁⚂⚃⚄⚅），6 歲孩子擲骰後直接對照
2. **圖示 + 文字並列**：
   - ⚀ 旁放拳頭圖 + 小字「力量」
   - ⚁ 旁放鞋子圖 + 小字「速度」
   - 以此類推
3. **數值放大**：屬性數值用 24pt 以上粗體，一眼可見
4. **顏色編碼**（選用）：
   - 力量＝紅色
   - 速度＝黃色
   - 攻擊＝橘色
   - 防禦＝藍色
   - 智慧＝紫色
   - 天賦＝金色

### 列印建議
- 紙張：160-200 gsm 厚磅紙
- 護貝後裁切，增加耐用度
- 或使用透明卡套 + 廢棄撲克牌當內襯

## 快速開始（第一場遊戲）

1. 讓孩子各選一張動物卡
2. DM 擲 d6 查開局情境表，說故事
3. 描述第一個挑戰（如：「前面有一條湍急的河流」）
4. 問孩子：「你們想怎麼過河？用誰的什麼能力？」
5. 孩子選擇後，擲 2d6 + 對應屬性
6. 根據結果推進故事
7. 累積 5 前進標記 → 進入終局 → 勝利！
