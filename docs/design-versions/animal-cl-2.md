# 《動物守護者 (Animal Guardians)》 MVP 遊戲設計文件 v1.1

> **整合版本**：`animal-cl-2`  
> **核心體驗**：由家長引導的輕策略合作 RPG  
> **適用對象**：6 歲（圖像直覺）、10 歲（數值策略）、家長（DM 敘事）

---

## Part 1: 給 DM（家長）的規則摘要卡 (One-Pager Rule)

### 1. 遊戲準備 (Setup)
*   **角色**：兩位玩家各選 1 張動物卡。
*   **生命**：桌子中央放置 **5 顆愛心**（隊伍共享）。
*   **目標**：收集 **5 枚前進標記** 後，完成終局挑戰（Boss 或逃脫）。

### 2. 核心流程 (The Loop)
1.  **遭遇**：DM 擲骰或口述一個情境（參見 Part 3）。
2.  **決策**：孩子們討論「派誰去？」與「用什麼能力？」。
3.  **檢定**：
    *   DM 確認對應屬性（1-6）。
    *   玩家擲 **2 顆骰子 (2d6)**。
    *   **判定值 = 擲骰總和 + 卡牌屬性數值**。
4.  **結算**：
    *   **成功 (≥ 12)**：獲得 1 枚前進標記，DM 描述帥氣的成功畫面。
    *   **失敗 (< 12)**：扣 1 顆愛心，DM 描述受挫但沒放棄的情境。
    *   **大成功 (雙 6)**：直接勝利！獲得 2 枚前進標記 + 1 枚「勇氣標記」（抵銷一次傷害）。
    *   **大失敗 (雙 1)**：直接失敗！扣 2 顆愛心。

### 3. 優勢與劣勢 (調整難度)
為降低運算負擔，採用調整**目標值 (DC)** 機制：
*   **標準難度**：**12**
*   **優勢 (Advantage) → DC 10**：
    *   屬性剋制（如：用水滅火）。
    *   孩子能生動描述合作方式（如：「哥哥用大象吸水，我用老鷹把水灑下去！」）。
*   **劣勢 (Disadvantage) → DC 14**：
    *   環境極端惡劣（如：在大雨中攀岩）。
    *   屬性明顯不合（如：用蠻力解謎題）。

## Part 3: 劇本與情境表 (Scenario Tables)

DM 每次擲 **1d6** 決定，或依故事邏輯選擇。

### 表 A：開局情境 (為何而戰？)
| d6 | 情境主題 | 任務描述 |
|:--:| :--- | :--- |
| **1** | **迷途搜救** | 暴風雨後，一隻幼獸被困在懸崖邊，隨時會掉下去！ |
| **2** | **守護水源** | 貪婪的野豬群霸佔了唯一的池塘，大家快渴死了。 |
| **3** | **森林大火** | 閃電引發火災，必須開出一條防火道保護家園。 |
| **4** | **神秘瘟疫** | 樹木變黑枯萎，要前往山頂尋找傳說中的「月光草」。 |
| **5** | **入侵危機** | 一群機械老鼠正在啃食森林的根基，必須破壞牠們的基地。 |

### 表 C：終局模式 (Boss 戰 / 逃脫戰)
當集滿 5 枚前進標記時觸發。

*   **模式 A：Boss 決戰 (硬碰硬)**
    *   連續 **3 輪** 戰鬥。
    *   每輪 Boss 會使出一種攻擊（DM 描述），難度皆為 **DC 14**。
    *   3 輪中只要 **成功 2 次** 即獲勝。

*   **模式 B：極限逃脫 (連續闖關)**
    *   連續 **3 關**，每關擲 d6 決定障礙。
    *   只要 **失敗 1 次** 就扣愛心並繼續跑；愛心歸零前跑完即獲勝。
    *   **逃脫障礙表 (d6)**：
        *   1 (力)：巨大倒木 / 厚重石門
        *   2 (速)：滾動巨石 / 斷裂吊橋
        *   3 (攻)：荊棘藤蔓 / 攔路小兵
        *   4 (防)：落石雨 / 瀑布沖刷
        *   5 (智)：謎題鎖 / 迷幻森林
        *   6 (技)：宿敵現身 (必須用 Slot 6 對決)

## Part 4: A4 版面設計指引 (Layout Guide)

為確保 6 歲孩童能獨立操作，採用 **「視覺與數值直連」** 的 UI 設計。

### 1. 頁面規劃
*   **規格**：A4 橫式，每頁 8 張卡片 (4 x 2 排列)。
*   **尺寸**：單卡約 63mm x 88mm (標準撲克牌)。
*   **材質**：建議使用 160gsm 以上紙張列印，或護貝。

### 2. 卡片 UI 結構 (由上而下)

```text
┌───────────────────────────────┐
│  [圖示] 獅子頭像 (40% 高度)    │
│  獅子  LION                   │  <-- 標題區
├───────────────────────────────┤
│  ┌────┐ ┌────┐ ┌────┐   │
│  │ ⚀拳 │ │ ⚁鞋 │ │ ⚂劍 │   │
│  │ ７ │ │ ５ │ │ ７ │   │
│  └────┘ └────┘ └────┘   │
│  ┌────┐ ┌────┐ ┌────┐   │  <-- 核心區 (Grid)
│  │ ⚃盾 │ │ ⚄腦 │ │ ⚅星 │   │      骰點 + 圖示 + 大數字
│  │ ５ │ │ ４ │ │10 │   │
│  └────┘ └────┘ └────┘   │
├───────────────────────────────┤
│  ★ 王者咆哮 (Roar)           │  <-- 技能區
└───────────────────────────────┘
```

### 3. 給 6 歲兒童的視覺動線 (UX)
1.  **聽指令**：DM 說「這關要比力氣，是 1 號！」(擲出 1 點)。
2.  **找骰子**：小孩在卡片上尋找 **⚀ (一點)** 的圖示。
3.  **看圖確認**：看到旁邊有 **拳頭** (Double Check)。
4.  **讀數字**：唸出下方的大數字 **7**。
5.  **運算**：拿起 2 顆骰子擲骰，加上 7。

### 4. 配色編碼 (建議)
*   ⚀ 力量：**紅色** (熱血、力量)
*   ⚁ 速度：**綠色** (風、敏捷)
*   ⚂ 攻擊：**橘色** (衝擊、爆炸)
*   ⚃ 防禦：**藍色** (堅固、冷靜)
*   ⚄ 智慧：**紫色** (神秘、思考)
*   ⚅ 天賦：**金色** (必殺技、特別)
