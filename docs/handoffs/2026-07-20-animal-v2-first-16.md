## 注音確認

- 「骰子」顯示 `ㄊㄡˊ ˙ㄗ` 是教育部《國語辭典簡編本》的正式讀音，不是字型或 PDF 錯誤。
- 日常常見 `ㄕㄞˇ ˙ㄗ`，但正式卡面不新增 override，維持 `ㄊㄡˊ ˙ㄗ`。

## 列印方式

1. 使用 A4 二分黑白雷射標籤紙。
2. PDF 維持 A4 橫式。
3. 列印縮放選 `100%`／「實際大小」，不要選「符合頁面」。
4. 建議使用 600 dpi；本 PDF 的動物圖有效解析度為 300 ppi，提高到 1200 dpi 只會略微改善文字／線條邊緣。
5. 沿卡面黑色外框裁切，再貼到放大版 Poker。

## 實體驗證清單

1. 量測裁切後是否為 `118.5×170mm`，並與實際 Poker 尺寸比對四周內縮。
2. 正常桌面距離閱讀動物名、屬性與技能文字，確認注音不糊成黑點。
3. 優先檢查長頸鹿淺灰動態尾跡、灣鱷腹面翻滾、紅毛猩猩抓握足與行軍蟻群體語意。
4. 用 16 張跑一次完整淘汰賽，記錄孩子是否能僅看圖說出動物及能力。
5. 根據試印結果決定是否調整圖片對比、卡面內縮量或個別圖片，再擴充剩餘 116 張。
