## 四、調整方案

### 方案 A：不調整（維持現狀）

- **優點**：數值平衡已驗證，無需重跑模擬
- **缺點**：智慧排序違反常識，可能被玩家質疑

### 方案 B：局部調整（維持總和 23-28）

只調整智慧屬性，用其他屬性補回總和：

| 動物 | 原數值 | 調整後 | 總和 |
|------|--------|--------|:----:|
| 大猩猩 | 力7/速4/攻5/防5/智7 | 力7/速4/攻**3**/防5/智**9** | 28 |
| 非洲象 | 力8/速2/攻6/防7/智5 | 力8/速**1**/攻**4**/防7/智**8** | 28 |
| 貓頭鷹 | 力2/速5/攻4/防3/智9 | 力2/速5/攻**5**/防**5**/智**6** | 23 |
| 加拉巴哥象龜 | 力4/速1/攻3/防8/智7 | 力**5**/速1/攻**5**/防8/智**4** | 23 |

- **優點**：修正最明顯問題，維持總和框架
- **缺點**：部分屬性變動較大（如大猩猩攻擊 5→3）

### 方案 C：全面調整

採納所有建議，允許總和超出 23-28 範圍。

- **優點**：最符合現實
- **缺點**：打破平衡框架，需重跑模擬驗證
